# Testing Guide

This directory contains comprehensive tests for the 3D Surface Reconstruction project.

## Test Structure

- `test_models.py` - Unit tests for neural network models
- `test_utils.py` - Tests for utility functions and data loaders  
- `test_training.py` - Tests for training components and loss functions
- `conftest.py` - pytest configuration and fixtures

## Running Tests

### Run All Tests

```bash
python run_tests.py

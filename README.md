# 3D Surface Reconstruction from Point Clouds

Deep learning pipeline for 3D surface reconstruction from point clouds using Signed Distance Functions (SDF).

## Features

- **PointNet-based feature extraction**
- **Multi-scale SDF prediction**
- **Efficient gradient computation**
- **Comprehensive metrics and visualization**
- **Production-ready code structure**

## Installation

```bash
pip install -r requirements.txt
python setup.py develop
